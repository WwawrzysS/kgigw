# Panel KGW

Prosta lokalna strona do porządkowania pracy Koła Gospodyń i Gospodarzy Wiejskich.

## Jak uruchomić

Otwórz plik `index.html` w przeglądarce.

## Co jest w pierwszej wersji

- Pulpit z podsumowaniem kasy, członków, zaległych składek i dokumentów.
- Lista członków.
- Składki członkowskie z oznaczeniem zapłacone/do zapłaty.
- Kasa: wpływy, wydatki, zakupy i zbiórki.
- Wydarzenia oraz kiermasze.
- Dokumenty i poczta jako miejsce na pisma, faktury, paragony i notatki z wiadomości.
- Zarząd koła.
- Wyszukiwanie w danych.
- Eksport i import danych do pliku JSON.

## Ważne

Dane zapisują się w przeglądarce na tym komputerze. Na kolejnym etapie można zrobić wersję z logowaniem, wspólną bazą danych i prawdziwym połączeniem z pocztą Gmail/Outlook.
